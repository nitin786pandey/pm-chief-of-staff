# Daily Briefs

Auto-generated artifacts from the three scheduled runs. Every file here was written by AI — never manually created.

## Folder structure

| Folder | Run | Cadence | Filename format |
|--------|-----|---------|----------------|
| `Morning Briefs/` | Morning Brief | 6 AM weekdays | `YYYY-MM-DD-Morning-Briefing.md` |
| `Evening Debriefs/` | Evening Debrief | 6–8 PM weekdays | `YYYY-MM-DD-Evening-Debrief.md` |
| `Weekly Radars/` | Weekly PM Radar | Saturday | `YYYY-MM-DD-Weekly-PM-Radar.md` |

Artifact discovery is filename-based — the validator and each scheduled run locate artifacts by matching the canonical filename pattern, not by title or content. A file with the wrong name won't be found.

Sample artifacts are in [`../01-PM-OS/`](../01-PM-OS/).
